# Public Assets

This directory contains static assets for the application.

## Required Files

### logo.png
- **Size**: 512x512px (recommended)
- **Format**: PNG with transparency
- **Usage**: Application logo displayed in navigation bar

### favicon.ico
- **Size**: 32x32px or 16x16px
- **Format**: ICO or PNG
- **Usage**: Browser tab icon

## Default Logo

A default SVG logo (`logo.svg`) is provided. To use it:
1. Convert `logo.svg` to PNG format
2. Save as `logo.png` in this directory
3. Create a favicon from the same design

## Custom Logo Creation

You can create your own logo using:
- Design tools (Figma, Adobe Illustrator, etc.)
- Online logo generators
- The provided SVG as a template

Make sure your logo:
- Has a transparent background
- Is readable at small sizes (for favicon)
- Matches your brand colors
- Works well on both light and dark backgrounds
